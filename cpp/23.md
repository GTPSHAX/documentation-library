# C++23

The current revision of the C++ standard.

# New language features

* New language feature testing macros
* , , a.k.a. "deducing `this`" ()
* Multidimensional subscript operator (e.g. `1=v[1, 3, 7] = 42;`) ()
* `static operator[]` (), `static operator()`, and static lambdas ()
* `auto(x)` and }: decay-copy in the language ()
* Assumptions via new attribute  ()
* Attributes on lambda-expressions ()
* Optional extended floating-point types: `std::float{16!`32|64|128}_t, and `std::bfloat16_t` ()
* New preprocessor directives: `#elifdef`, `#elifndef` (), and `cpp/preprocessor/error|#warning` ()
* Literal suffix '`Z`'/'`z`' for (`signed`) `std::size_t` literals (), e.g. `1=auto ouz = 0uz;`
* White-spaces trimming before line splicing. ()
* Simpler implicit move ()
* Extending the lifetime of temporaries in range-based for loop initializer ()
* CTAD from inherited constructors ()
* Labels at the end of compound statements (), e.g.<br>

## Notes

`<codecvt>` is deprecated in C++17 and removed in C++26 (see ).
The reason for removal is that this feature no longer implements the current [https://www.unicode.org/versions/latest/ Unicode Standard], supporting only the obsolete UCS-2 encoding.

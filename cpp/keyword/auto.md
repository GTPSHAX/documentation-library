## Usage

rev|until=c++11|
* Automatic storage duration specifier
rev|since=c++11|
* `auto` placeholder type specifier
* Function declaration with trailing return type
rev|since=c++17|
* Structured binding declaration
rev|since=c++20|
*
rev|since=c++23|
* Explicit cast to a prvalue copy

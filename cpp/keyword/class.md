## Usage

* declaration of a class
rrev|since=c++11|
* declaration of a scoped enumeration type
* In a template declaration, `class` can be used to introduce type template parameters and template template parameters
* If a function or a variable exists in scope with the name identical to the name of a class type, `class` can be prepended to the name for disambiguation, resulting in an .

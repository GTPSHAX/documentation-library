## Usage

rrev multi
|until1=c++11
|rev1=
Used to mark a template definition , which  allows the same template to be declared, but not defined, in other translation units.
|until2=c++20|rev2=
.
|rev3=
Marks a declaration, a group of declarations, or another module as exported by the current module.
